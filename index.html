OpenGL Anti Aliasing

Project Description

This is a project for COMP 371 Computer Graphics. The goal of this project is to demonstrate the use of anti-aliasing in OpenGL. The project will consist of a scene with multiple objects rendered to a display. The objects will be rendered with different anti-aliasing techniques enabled and disabled to show the difference in quality.
